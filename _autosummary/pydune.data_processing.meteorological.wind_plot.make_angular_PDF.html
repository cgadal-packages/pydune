Calculate the angular PDF (normalized) from input arrays.

Parameters:

  • angles (np.array) array of angles.

  • weight (np.array) array of weights. Its dimensions must match those of angles.

  • bin_edges (np.array) array containing the bins used to calculate the distribution (the default is np.linspace(0, 360, 361)).

  • axis (int) axis of the input aray along which the distribution is calculated (the default is -1).

Returns:

  • hist (np.array) array containing the distribution.

  • bin_centers (np.array) array containing the bin centers of the distribution.
